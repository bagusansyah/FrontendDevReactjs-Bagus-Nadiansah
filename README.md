# FrontendDevReactjs-Bagus Nadiansah

## Tech Stack
- React: 19.2.4
- React DOM: 19.2.4
- React Router DOM: 7.14.0
- MUI: 9.0.0
- Axios: 1.14.0
- React Scripts: 5.0.1
- JSON Server (dev): 1.0.0-beta.15
- Node.js: 18.x or later

## How to Start

Using npm:
```bash
npm install
npm start
```

Using yarn:
```bash
yarn install
yarn start
```

## Login
No authentication required.
